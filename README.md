# My development VScode profile (C++/Qt/CMake)

## How to Download and Import the VS Code Profile

1. **Download the Profile File**
	- Download the file `Qt.code-profile` from this repository.

2. **Import the Profile in VS Code**
	- Open Visual Studio Code.
	- Open the Command Palette (`Cmd+Shift+P` on macOS, `Ctrl+Shift+P` on Windows/Linux).
	- Type and select `Profiles: Import Profile...`.
	- Choose `Import from a file`.
	- Select the downloaded `Qt.code-profile` file.
	- Follow the prompts to finish importing.

This will set up your VS Code environment with the recommended settings and extensions for Qt development.
